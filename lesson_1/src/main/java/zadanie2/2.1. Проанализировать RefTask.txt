Результатом выполнения кода будет число 1, так как  при исполнении  метода change в главном методе  
новое значение переменной value не инициализируются, ввиду ссылочности типа Integer, т.е. Integer 
ссылается на Integer value = 1; соответственно его и печатает, изменения 
не распространяются за пределы метода change, чтобы получить желаемый результат число 2, необходимо
в метод change добавить строку System.out.println(value); Затем вызвать метод RefTask.change(value); 
в главном методе, удалив из главного метода строку System.out.println(value);
