1.Класс Item создает модели и переменные, которые будут использованы в классе Tracker.
2.Класс Tracker - класс где непосредственно создается хранилище для заявок через ArrayList и различными методами вносятся изменения в хранилище, затем после исполнения метода, снова отображается меню со списком действий.
3.Класс Сomment - класс используется для создания комментария, который добавляется/удаляется    методами addComm и delComm в классе Item.
4.Класс Start.UI-класс для запуска программы.
